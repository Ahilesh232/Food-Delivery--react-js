# Food-Delivery--react-js
FoodExpress

FoodExpress: A React + Vite project for an online food ordering and delivery platform. This app enables users to browse menus, place orders, and track deliveries with a fast, responsive interface powered by Vite's development server and React components. 1

Quick Start

Install dependencies: npm install

Run dev server: npm run dev (localhost:5173)

Build for production: npm run build
